ENGINEERED FEATURES



This directory contains datasets after the feature engineering stage.



FEATURE DESCRIPTION



The following specialized geomechanical features were calculated:



\- Vertical Projection: "Hourglass" shaped projection for tunnel influence.

\- Curvature: Boundary curvature of the excavated zones.

\- Density: Proximity-based density of excavated soil nodes.

\- Overlap Index: Interaction metric for twin tunnel scenarios.

\- Signed Distance: Normalized signed distance to the nearest excavation boundary.
