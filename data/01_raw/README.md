RAW PLAXIS DATA

This directory is intended to store raw simulation results exported from PLAXIS 2D via the automated scripting interface.

DATASET DESCRIPTION

- Format: CSV files.
- Content: Nodal results including coordinates (X, Y), soil status (Excavated/Non-excavated), and stress-strain parameters (SigxxE, SigyyE, Sigxy, Epsxx, Epsyy, Gamxy, Utot).
- Volume: ~1000 simulation cases (depending on the run).

DOWNLOAD INFORMATION

Due to the large total size, the full raw dataset is hosted externally.

- Link: https://disk.yandex.ru/d/EF5Qi17MxUe8FA
- Instructions: Download the archive and extract all .csv files directly into this folder.
